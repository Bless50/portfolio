StarterPay - API Middleware

Fintech Infrastructure

The Problem

Student developers in Cameroon are effectively locked out of the MTN MoMo payment ecosystem because business-only KYC requirements demand formal company registration beyond the reach of most student founders.

The Solution

A strategic API aggregator that sits between MTN's production APIs and students. By partnering with accredited institutions, it uses student enrollment as a secure KYC layer, allowing verified developers to launch production-ready apps while we manage security and compliance in the backend.

Goal

Bridging Student-to-Business KYC Gaps

Tech Why

Next.js and Node chosen for low-latency request forwarding and secure backend management of production keys.
